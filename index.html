Type some text into the text area on the left and it will appear on the whiteboard in the movement area

To put some text in bold, place asterisks on either side like so:

*This is in bold.*

To put some text in italics, place underscores on either side like so:

_This is in italics._

To change the colour of the text, write the name of the colour in curly brackets like so:

{red}This is in red.

To force a page break, type {pagebreak} like so:

This is on page 1.
{pagebreak}
This is on page 2.
